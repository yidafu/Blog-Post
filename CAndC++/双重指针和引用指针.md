 C++双重指针和引用指针

# 引言

最近写数据结构遇到了一个有趣问题的问题。

```c
class Tree{
private:
    Node* root;
public:
    Tree();
    void create( Node &TNode );
}

Tree::Tree() {
    this->root = NULL;
}

void Tree::create( Node &TNode ) {
    // 略
    this->create( TNode->childNode );
    this->create( TNode->nextSiling );
}

int main() {
    Tree test;
    // 为了简便起见
    test->create( this->root );
}

```

这样写在函数里建树成功了，但是，一返回main()函数所得的树就消失了。在 Stackoverflow 上找到了一个回答里面的文章很好的解释了这个问题。
